{# ── Roles & Access ── #}
Roles & Access
Preparer
  • Upload payroll / CSI files to start a new run
  • Re-upload files when a case is returned for correction
  • View case status and progress across all steps
  • Download generated check files and bank upload files
Reviewer
  • All Preparer access
  • Receive check approval request emails (Step 3 — first in chain)
  • Review check file data and approve or reject via email link
  • Return cases to Preparer with rejection reason if corrections needed
Approver
  • All Reviewer access
  • Receive final check approval emails (Step 3 — second in chain, after Reviewer)
  • Give final sign-off to release the accrual journal to Zoho Books
  • Triggers automatic bank file generation on approval
Arranger
  • Access to CSI module and Consultant Database only
  • Payroll module is not visible
  • Receives email notification whenever a CSI check has exception flags
  • Also notified when a CSI case is returned to the preparer for correction
  • Reviews flagged exceptions and updates consultant records to resolve them
Admin
  • Full access to all modules and cases
  • Invite and remove users, assign roles
  • View system settings and integration status
Note: Payment approval (Step 6) is sent to the Director and is handled separately via a secure email link — not through a system role.
